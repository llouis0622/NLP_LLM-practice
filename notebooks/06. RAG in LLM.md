# 1. 프롬프트 엔지니어링

- 자연어 처리에서 대규모 언어 모델과 효과적으로 상호작용하기 위해 프롬프트나 지시를 설계하는 기술
- 모델이 원하는 출력을 생성할 수 있도록 입력을 신중하게 구성하는 것 포함
- 언어 모델의 잠재력을 최대한 활용하고 사용자의 특정 요구에 부합하는 출력 획득 가능
- 프라이밍 : 특정 지시사항, 맥락, 원하는 결과와 일치하는 부분적인 문장을 포함시키는 것
- 시스템 프롬프트 : 모델에게 역할, 질문에 답하는 방식, 제약 조건 등 지시
- 사용자 프롬프트 : 프라이밍 단계에서 모델에게 보내지는 예시 프롬프트

# 2. RAG와 LangChain

## 1. RAG

- 대규모 언어 모델과의 원활한 상호작용을 위해 설계된 개발 프레임워크
- 텍스트 코퍼스가 사전 처리 과정을 거쳐 요약되거나 개별 청크로 분할된 후 벡터 공간에 임베딩
- 데이터 중 가장 관련성 높은 부분을 찾아내 이를 활용해 응답 생성

## 2. LangChain

- 임의의 대규모 언어 모델을 특정 데이터 소스와 연결 가능
- 도메인에 맞게 맞춤 설정 가능
- 특정한 처리 방식을 통해 데이터 처리를 빠르고 효율적으로 진행, 벡터 데이터베이스 생성
- 체인 : 프롬프트 템플릿, 다양한 데이터 소스에 대한 접근, 대규모 언어 모델에 대한 접근 등
- 에이전트 : 체인의 결과를 받아 처리 후 정해진 규칙에 따라 추가적인 분석과 의사결정 과정을 거쳐 더 심도 있는 결과 도출
